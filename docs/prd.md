# 需求文档

## 1. 应用概述

### 1.1 应用名称
AI私塾

### 1.2 应用描述
一款基于React Native开发的个人单机版跨平台移动学习应用，集成词本管理、试题练习、AI辅助学习、本地学习统计四大核心功能。支持词汇导入与AI生成学习内容、试卷识别与错题强化、AI口语陪练与写作批改、个人学习数据统计与排行榜激励。支持WebDAV/SMB/本地备份与恢复、学习数据分享、第三方词库导入、离线AI基础查询、多语言学习、学习提醒推送、学习报告导出、本地大模型导入功能。新增真实录音口语评测、OCR+AI试题解析、词本批量导入与发音播报、学习统计增强、今日计划、内置词汇包、多样化练习模式、美化分享卡片等功能。v13版本新增版本号自动同步、口语自由对话模式、AI自动生成词汇包、竞速/闯关练习模式、分享卡片自定义模板功能。界面采用现代清爽风格，强调高可读性与专注学习体验。

### 1.3 技术参考
- 前端框架：React Native + Expo SDK 55 + NativeWind v4 + expo-router + TypeScript
- 本地数据库：SQLite
- AI大模型API：用户自行配置（DeepSeek/GPT-4o/通义千问等）
- 本地大模型推理：支持llama.cpp/MLX等引擎，用户可自行下载GGUF格式模型文件并导入
- 语音评测：科大讯飞或Google Speech
- 语音录音：expo-audio
- 语音识别：百度短语音识别skill（short-speech-recognition）
- 语音播报：expo-speech（系统TTS）
- 试卷OCR：用户可配置OCR服务API（默认Tesseract，支持百度/阿里OCR等）
- 版本号管理：expo-constants（从Constants.expoConfig?.version读取）
- 数据隐私：学习数据不上传第三方，调用API时脱敏处理
- 页面安全区域：使用react-native-safe-area-context的SafeAreaView
- 启动页控制：使用expo-splash-screen
- 文件选择：使用expo-document-picker
- 截图分享：使用react-native-view-shot + expo-sharing

### 1.4 界面风格
主题名称：「专注白板」（Clean Slate）
- 主色调：深蓝#2C5F8A或墨绿#2E6B5C
- 强调色：暖橙#E67E22
- 背景色：浅灰白#F8F9FA（日间）/深灰#1E1E1E（夜间）
- 圆角：卡片12px，按钮8px
- 图标：Feather/Lucide风格
- 间距：8pt网格系统

## 2. 用户与使用场景

### 2.1 目标用户
个人自用学习者，需要系统化管理词汇学习、试题练习，并借助AI提升学习效率。

### 2.2 核心使用场景
1. 导入词汇列表，通过AI生成音标、释义、例句，进行单词卡学习与艾宾浩斯复习
2. 上传试卷文件，AI识别题型并生成题库，练习后针对错题生成相似题强化训练
3. 使用AI口语陪练进行跟读评测，获得纠音改错反馈；提交写作内容获得语法纠错与批改建议
4. 查看每日学习统计数据，通过个人排行榜进行自我激励
5. 将学习数据通过WebDAV/SMB/本地方式备份，更换设备后手动恢复数据
6. 将学习成绩生成分享卡片图片，分享至社交平台
7. 从第三方平台导出词库/题库文件后手动导入
8. 在无网络环境下使用离线AI模型进行基础单词查询和简单语法纠错
9. 切换目标学习语言，学习多种语言
10. 设置每日学习提醒，按时接收推送通知
11. 导出学习统计数据为PDF或Excel格式
12. 自定义首页励志文案，支持在线拉取文案库并定时/手动切换
13. 配置在线AI服务参数，切换不同AI服务
14. 切换夜间模式或跟随系统主题
15. 导入本地大模型（GGUF格式），在部分功能中优先使用本地推理
16. 使用真实录音进行口语评测，自动语音转文字并AI评分
17. 上传试卷后OCR识别并AI结构化解析题目
18. 批量导入CSV格式单词，使用TTS播报单词发音
19. 查看连续打卡天数、累计学习时长等成就指标
20. 设置今日学习计划并跟踪完成进度
21. 从内置词汇包快速导入常用词汇
22. 选择顺序/随机/限时挑战等多种练习模式
23. 生成美化的打卡分享卡片
24. 使用口语自由对话模式与AI进行多轮对话练习
25. 通过AI自动生成指定主题的词汇包
26. 使用竞速模式或闯关模式进行试题练习
27. 自定义分享卡片模板内容与显示项

### 3.2 功能说明

#### 3.2.1 启动页（SplashScreen）
- 展示应用图标居中、纯色背景的自定义启动屏
- 在根_layout.tsx中使用expo-splash-screen控制启动页显示与隐藏
- 加载完成后隐藏启动页，进入首页
- 消除白屏闪现问题

#### 3.2.2 首页（学Tab）
- 使用SafeAreaView包裹页面内容，避免与状态栏/导航栏重叠
- 展示今日学习摘要卡片：学习时长、新词数、正确率、试题数
- 标题栏下方展示励志文案，支持点击切换
- 四大快捷入口：词本管理、试题练习、AI学习、学习统计
- 艾宾浩斯复习提醒卡片：展示今日需复习的单词数量，点击进入复习页
- 今日计划卡片：展示今日学习目标（复习X个单词 + 做Y道题）、当前完成进度、进度条可视化展示、完成目标时显示庆祝效果、点击卡片可进入设置页修改目标

#### 3.2.3 词本管理模块

**词本列表页**
- 使用SafeAreaView包裹页面内容
- 展示词本卡片：名称、单词数量、学习进度条
- 环形进度条显示已掌握/总单词数，可视化掌握进度
- 每个词本右侧：学习按钮、更多菜单（编辑名称、导出CSV、删除词本）
- 支持新建词本

**词本导入页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- 支持上传TXT/CSV文件
- 支持直接粘贴文本内容
- 支持第三方文件导入（CSV/APKG/TXT文件）
- 批量导入CSV：支持CSV格式文件（每行：单词,音标,释义,例句），用户选择目标词本后批量导入，导入后展示导入成功数量
- 内置词汇包：提供4个内置词汇包（英语四级高频词50个词样例、英语六级高频词50个词样例、日语N5基础词汇50个词样例、英语商务词汇30个词样例），用户选择词汇包后一键导入到指定词本
- **AI生成词汇包（新增v13）**：
  - 新增「AI生成词汇包」入口
  - 用户输入主题（如「旅行常用词」「科技词汇」）和目标数量
  - 调用AI服务生成对应词汇列表（英语单词+中文释义格式）
  - 展示生成结果预览页，用户可编辑后一键导入词本
- 导入后AI自动生成音标、释义、例句

**单词卡学习页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- 以卡片形式展示单词，包含单词、音标、释义、例句
- 每个单词卡片右上角增加发音按钮，点击调用expo-speech播报单词
- 支持左右滑动切换单词
- 支持标记「已掌握」或「未掌握」
- 支持编辑当前单词：可修改单词、音标、释义、例句
- 支持删除当前单词：长按卡片弹出确认框
- 词本详情页增加「导出为CSV」按钮

**AI助记故事页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- 在单词卡学习页每个单词卡片下方增加「生成故事」按钮
- 点击后跳转或弹窗显示AI生成的助记故事

**艾宾浩斯复习页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- 页面上方增加「选择词本」下拉框，支持选择单个词本或「全部词本」
- 根据艾宾浩斯遗忘曲线（1天、2天、4天、7天、15天）自动推送需复习的单词
- 展示复习单词列表，支持进入单词卡学习
- 每个单词卡片右上角增加发音按钮，点击调用expo-speech播报单词
- 标记「已掌握」的单词进入下一复习周期；标记「未掌握」的单词重置为第1天

#### 3.2.4 试题练习模块

**题库列表页**
- 使用SafeAreaView包裹页面内容
- 展示已导入的试卷列表：试卷名称、题目数量、上次练习进度
- 每个试卷右侧：继续练习按钮、更多菜单（重置进度、删除试卷）
- 支持新建题库

**试题导入页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- 支持上传PDF/图片/Word格式试卷文件
- OCR+AI解析：上传文件后调用用户已配置的OCR服务识别文本内容，识别出的文本送入已配置的AI服务，AI按JSON格式解析出完整题目列表（题目内容、选项、答案、解析），展示解析结果预览页，用户可编辑后保存
- 若识别失败或结果不准确，用户可进入题目编辑器手动修正
- 支持从第三方平台导出的题库文件手动导入

**练习模式选择页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- 在试卷详情/开始练习时展示
- 提供5种练习模式：
  - 顺序模式：按原有顺序练习
  - 随机模式：题目随机打乱顺序
  - 限时挑战：用户设置总时间，超时自动提交，显示倒计时
  - **竞速模式（新增v13）**：所有题目连续作答，记录总完成时间和答题速度（题/分钟），完成后显示速度排名评级（S/A/B/C）
  - **闯关模式（新增v13）**：每答对5题为一关，连续答错2题则「闯关失败」弹出提示，可选择重新挑战或直接结束
- 用户选择模式后进入试题练习页

**试题练习页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- 展示题目内容，支持答题
- 支持标记对错
- 自动保存当前试卷的答题进度
- 再次进入同一试卷时，弹窗提示「是否继续上次练习？」
- 练习完成后，将试题中的生词自动加入当前活跃词本

**错题强化页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- 展示标记为错误的题目列表
- AI针对错题生成相似题目，用于强化训练
- 每个错题最多生成5道相似题，系统记录已生成题目的内容哈希值，避免重复
- 用户可点击「再生成一道」追加生成（不超过上限）
- 支持重新练习相似题
- 增加「导出错题集」按钮，导出错题内容为CSV或JSON格式
- 错题专项练习：新增「错题专项练习」按钮，点击后仅练习标记为错误的题目，支持选择练习模式（顺序/随机/限时/竞速/闯关）

#### 3.2.5 AI学习模块

**口语陪练页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- **跟读评测模式**：
  - App给出预设句子或用户选择的句子
  - 用户点击录音按钮，使用expo-audio开始录音
  - 录音完成后，调用百度短语音识别skill（short-speech-recognition）将语音转为文字
  - 转文字后自动与目标句子对比，送入AI评测
  - 若未配置百度短语音识别skill，降级为手动输入文本模式
  - 调用科大讯飞或Google Speech进行语音评测
  - AI提供评分和纠音改错反馈（按单词/音素标红）
  - 无网络环境下，若已下载离线AI模型，调用离线模型提供基础反馈
- **自由对话模式（新增v13）**：
  - 在页面新增Tab或入口切换至自由对话模式
  - 用户按住录音按钮说话，松开后触发语音识别（调用已有Edge Function）
  - 识别文字发送给AI（使用已有aiService.ts）进行多轮对话
  - AI回复文字后用expo-speech TTS朗读出来
  - 支持切换对话主题（日常聊天、旅行、购物、面试等）
  - 显示完整对话历史记录
  - 支持清空对话历史

**语法纠错页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- 用户输入文本内容
- AI识别语法错误并提供纠正建议
- 支持选择使用本地大模型或在线API（页面内切换）
- 无网络环境下，若已导入本地大模型且启用，使用本地推理

**写作批改页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- 用户提交写作内容（建议不超过500词）
- AI提供批改意见，包括语法、用词、结构等方面的建议
- 此功能强制使用在线API

**动态复习推荐页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- 页面上方增加「选择词本」下拉框，支持选择单个词本或「全部词本」
- 基于词本中单词的掌握度，AI动态推荐需复习的单词
- 优先推荐未掌握或复习次数较少的单词

#### 3.2.6 学习统计模块

**统计数据页**
- 使用SafeAreaView包裹页面内容
- 展示学习统计图表（柱状图或折线图），支持按周/月切换
- 记录并展示每日学习数据：学习时长、新词数、正确率、试题数
- 支持按日期查看历史统计数据

**个人排行榜页**
- 使用SafeAreaView包裹页面内容
- 生成个人排行榜，包含以下维度：
  - 周最高学习量（过去7天内单日最长学习时长）
  - 月最高学习量（过去30天内单日最长学习时长）
  - 周最佳正确率（过去7天内单日最高正确率）
  - 月最佳正确率（过去30天内单日最高正确率）
- 若无足够数据，显示「数据不足，继续学习解锁排行榜」

**成就指标卡片**
- 使用SafeAreaView包裹页面内容
- 在个人中心（profile.tsx）的统计区域新增成就感指标卡片
- 展示以下指标：
  - 连续打卡天数（streak）
  - 累计学习总时长（分钟）
  - 累计单词总数
  - 累计答题总数
- 数据从daily_stats表聚合计算

**学习数据分享页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页
- 支持将学习统计、排行榜成绩等数据生成分享卡片
- 使用react-native-view-shot将分享卡片区域截图为PNG图片
- 通过expo-sharing分享图片文件
- 美化分享卡片：增加连续打卡天数显示、增加学习语言徽章、提供3种卡片主题色选择
- **自定义模板（新增v13）**：
  - 在分享页面增加编辑功能
  - 支持修改卡片标题文字（默认「今日学习战报」）
  - 支持修改底部签名文字（默认「AI私塾 · 专注学习」）
  - 支持显示/隐藏各数据项（学习时长、单词数、正确率、试题数）
  - 编辑完成后实时预览效果
- 分享卡片包含学习时长、新词数、正确率、排行榜成绩等信息

**学习报告导出页**
- 使用SafeAreaView包裹页面内容
- 返回按钮使用router.back()回到上一页

## 4. 业务规则与逻辑

### 4.1 版本号自动同步规则（新增v13）
- src/lib/version.ts中的版本号从expo-constants读取（Constants.expoConfig?.version）
- 确保版本号永远与app.json保持一致，不再需要手动维护

### 4.2 词本导入与AI生成规则
- 用户上传TXT/CSV文件或粘贴文本后，系统提取单词列表
- 调用用户配置的在线AI服务生成音标、释义、例句
- 生成内容保存至本地数据库（SQLite）
- 若未配置在线AI服务，提示用户配置

### 4.3 第三方词库导入规则
- 用户从第三方平台自行导出词库文件（CSV/APKG/TXT）
- 系统解析文件内容并保存至本地数据库
- 导入后触发AI生成音标、释义、例句（需联网）

### 4.4 批量导入CSV规则
- 用户在词本导入页选择「导入CSV」
- 上传CSV文件（格式：单词,音标,释义,例句）
- 系统解析CSV内容，批量导入至指定词本
- 导入完成后展示成功数量

### 4.5 内置词汇包规则
- 用户在词本导入页选择「内置词汇包」
- 展示4个内置词汇包列表
- 用户选择词汇包后，系统将词汇包内容导入至指定词本
- 导入完成后展示成功数量

### 4.6 AI生成词汇包规则（新增v13）
- 用户在词本导入页选择「AI生成词汇包」
- 用户输入主题（如「旅行常用词」「科技词汇」）和目标数量
- 调用AI服务生成对应词汇列表（英语单词+中文释义格式）
- 展示生成结果预览页，用户可编辑后一键导入词本

### 4.7 单词发音播报规则
- 用户在单词卡学习页或艾宾浩斯复习页点击发音按钮
- 系统调用expo-speech播报当前单词
- 播报语言根据当前学习语言自动适配

### 4.8 词本掌握进度可视化规则
- 词本列表页每个词本卡片显示环形进度条
- 进度条展示已掌握单词数/总单词数
- 实时更新进度数据

### 4.9 试题导入与识别规则
- 用户上传PDF/图片/Word文件后，调用OCR服务识别文本
- AI识别题型（选择题、填空题、判断题等），提取题目内容与选项
- 若识别失败或准确率低，用户可进入题目编辑器手动修正
- 生成题库保存至本地数据库

### 4.10 OCR+AI解析规则
- 用户上传试卷文件后，调用用户已配置的OCR服务识别文本
- 识别出的文本送入已配置的AI服务
- AI按JSON格式解析出完整题目列表（题目内容、选项、答案、解析）
- 展示解析结果预览页，用户可编辑后保存至题库

### 4.11 练习模式规则
- 用户在试卷详情/开始练习时进入练习模式选择页
- 顺序模式：按原有顺序练习
- 随机模式：题目随机打乱顺序
- 限时挑战：用户设置总时间（分钟），超时自动提交，页面显示倒计时
- **竞速模式（新增v13）**：所有题目连续作答，记录总完成时间和答题速度（题/分钟），完成后显示速度排名评级（S/A/B/C）
- **闯关模式（新增v13）**：每答对5题为一关，连续答错2题则「闯关失败」弹出提示，可选择重新挑战或直接结束
- 用户选择模式后进入试题练习页

### 4.12 错题专项练习规则
- 用户在错题强化页点击「错题专项练习」按钮
- 系统筛选标记为错误的题目
- 用户选择练习模式（顺序/随机/限时/竞速/闯关）
- 进入试题练习页，仅练习错题

### 4.13 试题练习与生词加入规则
- 用户完成试题练习后，系统自动提取试题中的生词（基于词本库判断是否为新词）
- 生词自动加入当前活跃词本，并触发AI生成音标、释义、例句

### 4.14 错题强化规则
- 用户标记错题后，AI根据错题内容和正确答案生成相似题目
- 每个错题最多生成5道相似题，系统记录已生成题目的内容哈希值，避免重复
- 相似题保存至错题强化题库，供用户重新练习
- 用户可点击「再生成一道」追加生成（不超过上限）

### 4.15 艾宾浩斯复习规则
- 根据艾宾浩斯遗忘曲线（1天、2天、4天、7天、15天），自动推送需复习的单词
- 用户标记「已掌握」的单词进入下一复习周期
- 用户标记「未掌握」的单词，重置复习计时为第1天，重新开始周期

### 4.16 AI动态复习推荐规则
- 基于词本中单词的掌握度（已掌握/未掌握标记、复习次数、复习间隔），AI动态推荐需复习的单词
- 优先推荐未掌握或复习次数较少的单词

### 4.17 口语陪练录音与语音识别规则
- 用户在口语陪练页点击录音按钮，使用expo-audio开始录音
- 录音完成后，调用百度短语音识别skill（short-speech-recognition）将语音转为文字
- 转文字后自动与目标句子对比，送入AI评测
- 若未配置百度短语音识别skill，降级为手动输入文本模式
- AI提供评分和纠音改错反馈

### 4.18 口语自由对话规则（新增v13）
- 用户在口语陪练页切换至自由对话模式
- 用户按住录音按钮说话，松开后触发语音识别（调用已有Edge Function）
- 识别文字发送给AI（使用已有aiService.ts）进行多轮对话
- AI回复文字后用expo-speech TTS朗读出来
- 支持切换对话主题（日常聊天、旅行、购物、面试等）
- 显示完整对话历史记录
- 支持清空对话历史

### 4.19 离线AI模型规则
- 用户在有网络环境下下载轻量级离线AI模型（大小控制在100MB以内）
- 无网络环境下，系统调用离线AI模型实现基础功能：
  - 单词查询（音标、释义、简单例句）
  - 短句语法纠错（不超过20个单词）
- 离线AI模型不支持：自由对话、长篇写作批改、复杂题型生成
- 若功能需要联网而网络不可用，提示用户「此功能需要网络连接」

### 4.20 多语言学习规则
- 用户在语言切换页选择目标学习语言
- 系统根据所选语言调整词本、题库、AI学习等功能的语言适配

- 新词数：记录当日新增至词本的单词数量
- 正确率：记录试题练习中标记正确的题目占比（%）
- 试题数：记录当日完成的试题数量（题目计数，非卷数）

### 4.32 个人排行榜规则
- 周最高学习量：统计过去7天内单日最高学习时长（分钟）
- 月最高学习量：统计过去30天内单日最高学习时长（分钟）
- 周最佳正确率：统计过去7天内单日最高正确率（%）
- 月最佳正确率：统计过去30天内单日最高正确率（%）
- 若无足够数据（如学习天数不足），显示「数据不足，继续学习解锁排行榜」

### 4.33 数据隐私规则
- 学习数据（词本、题库、统计数据）仅存储于本地数据库（SQLite），不上传第三方服务器
- 调用AI API时，对可能包含个人隐私的文本进行脱敏处理（如移除姓名、手机号等）
- 备份数据仅用于用户个人恢复，App不会主动上传至任何云端
- API Key等敏感信息在本地加密存储

### 4.34 本地大模型使用规则
- 用户自行下载符合llama.cpp标准的GGUF格式模型文件，放置于手机存储中，通过expo-document-picker选择并导入
- App不提供模型下载，仅提供文件选择和加载接口
- 本地大模型启用后，以下AI功能可切换使用本地推理：
  - 语法纠错（短句）
  - 单词释义与例句生成
- 以下功能因性能或精度原因，不支持本地模型（强制使用在线API）：
  - 长篇写作批改（>200词）
  - 复杂题型生成（相似题、助记故事等）
  - 口语自由对话
- 若用户启用本地模型但设备内存不足或推理失败，系统自动回退至在线API，并弹出提示
- 本地模型文件仅存储于用户设备，不会上传或分享
- 在「本地大模型配置页」增加推理优先级设置，可选：
  - 优先本地大模型>离线AI模型>在线API
  - 仅使用在线API
  - 用户可单独开关离线AI模型和本地大模型的启用状态
- 连续3次推理失败自动禁用本地大模型

### 4.35 页面返回逻辑规则
- 所有子页面（词本详情、单词卡学习、艾宾浩斯复习、导入词本、试题练习、导入试题、错题强化、AI语法纠错/写作批改/口语陪练/复习推荐、统计分享/导出、各设置子页）的返回按钮使用router.back()回到上一页，而非首页

### 4.36 页面安全区域规则
- 所有页面使用SafeAreaView（来自react-native-safe-area-context）包裹页面内容，避免内容与顶部状态栏/底部导航栏重叠

### 4.37 启动页控制规则
- 在根_layout.tsx中使用expo-splash-screen控制启动页显示与隐藏
- 启动页展示应用图标居中、纯色背景的自定义启动屏
- 加载完成后隐藏启动页，进入首页
- 消除白屏闪现问题

### 4.38 测试连接规则
- WebDAV备份配置、SMB备份配置、在线AI服务配置、OCR服务配置、首页文案API配置均增加「测试连接」按钮
- 点击「测试连接」按钮后，显示loading状态
- 连接成功：显示绿色「连接成功」提示
- 连接失败：显示红色错误原因

### 4.39 分享卡片生成规则
- 在学习数据分享页，使用react-native-view-shot将分享卡片区域截图为PNG图片
- 通过expo-sharing分享图片文件至社交平台

## 6. 验收标准

1. 用户在src/lib/version.ts中查看版本号，版本号从expo-constants读取（Constants.expoConfig?.version），与app.json保持一致
2. 用户在词本导入页选择「AI生成词汇包」，输入主题和目标数量，AI成功生成词汇列表，展示预览页后一键导入词本
3. 用户在试卷详情/开始练习时进入练习模式选择页，选择竞速模式后进入试题练习页，完成所有题目后显示总完成时间、答题速度和速度排名评级（S/A/B/C）
4. 用户在试卷详情/开始练习时进入练习模式选择页，选择闯关模式后进入试题练习页，每答对5题为一关，连续答错2题则「闯关失败」弹出提示，可选择重新挑战或直接结束
5. 用户在口语陪练页切换至自由对话模式，按住录音按钮说话，松开后触发语音识别，识别文字发送给AI进行多轮对话，AI回复文字后用TTS朗读出来
6. 用户在口语陪练页自由对话模式中切换对话主题（日常聊天、旅行、购物、面试等），显示完整对话历史记录，支持清空对话历史
7. 用户在学习数据分享页进入编辑功能，修改卡片标题文字、底部签名文字，显示/隐藏各数据项（学习时长、单词数、正确率、试题数），编辑完成后实时预览效果
8. 用户在所有页面中，页面内容使用SafeAreaView包裹，内容不与顶部状态栏/底部导航栏重叠
9. 用户启动App时，展示图标居中、纯色背景的自定义启动屏，加载完成后隐藏启动页进入首页，无白屏闪现
